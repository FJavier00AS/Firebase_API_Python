PASOS PARA USAR ESTA API:

1. Crea un proyecto en Firebase Console.
2. Activa Firebase Realtime Database y Authentication (correo/contraseña).
3. Descarga las credenciales de servicio (archivo JSON) y guárdalo como:
   firebase_credentials.json
4. Cambia la URL de database en main.py (donde dice TU_PROYECTO.firebaseio.com)
5. Instala dependencias: pip install -r requirements.txt
6. Ejecuta: python main.py

Opcional: Sube a Render.com, Railway.app o Fly.io
